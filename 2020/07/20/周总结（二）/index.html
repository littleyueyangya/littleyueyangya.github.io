文献阅读-计算机视觉

因为极市开发者社区平台上已经对计算机视觉进行了大致的分类，这周的工作是把各个小方向的综述大致看了一遍。

看文献的时候带着这几个问题进行

1）Describe what the authors of the paper aim to accomplish, or perhaps did achieve.描述这篇论文的作者想要完成什么，或者已经完成了什么。（即Motivation和创新）

2）If a new approach/technique/method was introduced in a paper, what are the key elements of the newly proposed approach?如果在一篇论文中介绍了一种新方法/技术/方法，该新方法的关键要素是什么?

（新方法的与以往方法的不同之处，实现该方法的关键步骤）

3）What content within the paper is useful to you?论文中哪些内容对你有用?

4）What other references do you want to follow?你还想关注哪些参考资料/文献？

1、人脸识别

《Face Recognition Using Neural Network: A Review》

1）从数据库或视频中识别人脸，对基于人工神经网络的人脸识别技术进行了总结，分析了相关技术的优缺点，讨论了神经网络如何应用于人脸识别系统，以及它如何比其他方法更有效。

2）因为是综述性文献，没有主要创新技术

3）目前人脸识别技术和系统发展现状，人脸识别系统的实现流程

4）因为这篇文章是2016年的，有点距离了，还想关注更近的文章，以及处理人工神经网络，还有没有其他技术用于人脸识别。

2、目标检测

《Object Detection With Deep Learning: A Review》—2019

目标检测定义：确定目标在给定图像中的位置(目标定位)和每个目标所属的类别(目标分类)。

1）针对几个特定的检测任务：通用对象检测，突出目标检测、人脸检测和行人检测。分别进行一些方法和改进策略进行综述和比较。并且提出了一些未来目标检测的发展方向。

2）综述

3）对目标检测的定义，主要的完成的任务，未来的方向有了一定的了解

4）可以对未来发展方向哪一块再细看一下

5、图像分类

《Deep Convolutional Neural Networks for Image Classification: A Comprehensive Review》

1）对基于深度卷积神经网络的图像分类方法进行综述

3）图像分类是其他计算机视觉任务的基础，比如定位、检测和分割(Karpathy, 2016)。还有图像分类未来的发展方向（第6节），描述了多标签分类和图像描述，该方向是计算机视觉和自然语言处理交叉部分（即此外，用正确的自然语言句子正确描述图像语义内容的能力是一个具有挑战性的问题(Vinyals, Toshev, Bengio， &Erhan, 2015)，发现自己在计算机视觉和自然语言处理之间的交叉部分）

文中提到的关于图像分类未来的研究方向

  1. 理论论证和内部理解。尽管DCNNs取得了经验上的成功，但缺乏能够证明其成功的理论依据。
  2. 几何不变性。尽管DCNNs具有抗小尺度变形的鲁棒性，它们最终的表示不是几何不变的，具体来说，它们对全局转换、旋转和缩放很敏感。因此，一个有趣的方向是调查是否需要对DCNN体系结构进行进一步的根本改变，以提高其普遍的鲁棒性。
  3. 面向移动部署。尽管有一些计算成本、内存占用和存储减少，在这方面改进DCNNs仍是一个开放的研究领域，尤其是在fpga、嵌入式系统、移动设备和其他内存和电池限制有限的设备上部署DCNNs。 相关进展包括DCNN压缩和权值量化、快速算法、截断表示的GPU集群和FPGA加速进展等。
  4. Deep Flaws. 分类器对于扰动敏感，当使用这些扰动来改变图像时，人类可以很容易地对图像进行正确的分类，而分类器将图像视为来自不同的类。方法一般是对抗性训练(Goodfellow et al.， 2015)和精馏（ distillation ），生成式预处理方法，例如使用去噪自动编码器（Uli cný等人，2016），并更改了DCNN架构，使其更加非线性或对异常信号进行了惩罚。
  5. 多标记图像和更深层次的图像理解。描述了使用自然语言处理方法RNN结合来实现；另一个有前景的方向是使用强化学习来训练这些组合架构。
  6. 网络模型超参数选择问题（一种可能的解决方法是使用演化算法，如粒子群优化(Kennedy & Eber- hart, 1995)，它已经成为一种流行的优化技术，进行超参数搜索，然后将结果与DCNNs集成。）；在线机器人系统中利用DCNNs的鉴别和表达分类能力；另一个趋势是卷积的因式分解以提高计算效率（如 Inception）；最大的挑战是缩小生物神经网络和DCNNs之间的理论差距。

《Computer vision based 3D reconstruction : A review》

三维重建在许多领域的应用都是从目标重建开始的，如场地、地面和海平面下的文物、医学影像数据、核物质等。

图像分类CVPR文献阅读

《Learning from Web Data with Self-Organizing Memory Module》

1）用于解决web数据的分类标签噪声和背景噪声。并利用记忆模块处理多实例学习框架下web数据的标签/背景噪声;在训练阶段只使用有噪声的web图像，可以同时解决标签噪声和背景噪声。

2）关键技术是Self-Organizing Memory Module，思想源于 Self-Organizing Map [48] 方法

3）其中处理web数据的噪声分类，和处理web数据中存在的问题对我有帮助。原有的web数据主要关注于标签噪声，其中研究主要关于三个方向，分别是消除标签噪声，鲁棒模型及 curriculum learning。

4）无

《Making Better Mistakes: Leveraging Class Hierarchies with Deep Networks》

1）对错误进行分级描述，因为目前的图像分类模型中，将所有判断为非真实类的错误当作是相同程度的错误。但是有些情况下并非如此。该文献的目的是区分错误的之间的等级，争取犯更小的错误。“就目前而言，似乎有可能犯更好的错误，但定义更好概念的阶级关系的性质是至关重要的。”
